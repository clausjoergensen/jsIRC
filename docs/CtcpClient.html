CtcpClient

CtcpClient

Represents a client that communicates with a server using CTCP (Client to Client Protocol), operating over an IRC connection.

Constructor

new CtcpClient(client)

Constructs a new CtcpClient for a given IrcClient.

Example
let ircClient = new IrcClient()
let ctcpClient = new CtcpClient(ircClient)

ctcpClient.on('version', (source, versionInfo) => {
  console.log(`[${source.nickName} VERSION reply]: ${versionInfo}.`)
}

ctcpClient.version(['Rincewind'])
Parameters:
Name Type Description
client IrcClient

The IRC client by which the CTCP client should communicate.
